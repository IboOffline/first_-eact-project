# context.md — Übungswebseite „Über mich“

## Projektziel
Eine persönliche Landingpage, die mich als Person, Entwickler und kreativen Kopf vorstellt.  
Sie dient als digitale Visitenkarte und Portfolio-Basis.

## Technischer Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Fonts:** Poppins (Headlines), Inter (Body)
- **Farbschema:** Definiert in `tailwind.config.js` (`brand`, `accent`)
- **Struktur:** Eine einzelne Landingpage mit scrollbaren Abschnitten
- **Deployment:** Manuell über GitHub → Server

## Seitenstruktur
1. **Hero / Intro**
   - Großes Profilbild oder Illustration
   - Headline mit Name („Hi, ich bin Ibrahim Can Uyan“)
   - Untertitel: „Webentwickler & Kreativer Denker“
   - CTA-Button („Kontaktiere mich“ / „Mehr erfahren“)

2. **Über mich**
   - Kurzbiografie: Wer ich bin, woher ich komme, was mich motiviert
   - Meine Werte: Lernen, Kreativität, Selbstständigkeit

3. **Fähigkeiten & Arbeit**
   - Tech Stack (HTML, CSS, JavaScript, React, Laravel, Tailwind, GitHub, Figma)
   - Arbeitsweise und Philosophie
   - Optional: Timeline oder Grid für Skills

4. **Projekte / Referenzen**
   - Drive More – Website für KFZ-Meisterbetrieb
   - Soleia Skincare – E-Commerce Pre-Order Konzept
   - CanBeOnline – Digitale Agentur / Portfolio
   - (jeweils mit kurzem Text und Vorschaubild)

5. **Hobbys & Persönlichkeit**
   - Themen: Reisen, Musik, Drohnenfotografie, Design
   - Darstellung: Icons, kurze Sätze oder Bilder

6. **Kontakt & Footer**
   - Kontaktmöglichkeit (E-Mail, Socials)
   - Footer mit Impressum, ggf. © Jahr + Name

## Design-Richtlinien
- Viel Weißraum, sauberes Layout
- Farbkontrast: Weiß (#fff) und brand-500 (#4d7dff)
- Headlines in **Poppins**, Fließtext in **Inter**
- Buttons mit abgerundeten Ecken, Hover-Effekte über Tailwind

## Zukünftige Erweiterungen
- Mehrsprachigkeit (Deutsch/Englisch)
- Animierte Sektionen (Framer Motion / Scroll Animation)
- Kontaktformular mit Node-Mailer oder Formspree
- SEO-Optimierung (Meta, OpenGraph, Schema.org)

---

> **Ziel:** Eine minimalistische, glaubwürdige „Über mich“-Seite, die sowohl Persönlichkeit als auch Professionalität ausdrückt und als Grundlage für spätere Portfolio-Projekte dient.
